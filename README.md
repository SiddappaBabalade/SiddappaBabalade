![Java](https://img.shields.io/badge/Java_21-ED8B00?style=flat-square&logo=openjdk&logoColor=white)
![Spring Boot](https://img.shields.io/badge/Spring_Boot_3.5-6DB33F?style=flat-square&logo=spring-boot&logoColor=white)

![JWT](https://img.shields.io/badge/JWT-000000?style=flat-square&logo=json-web-tokens&logoColor=white)

**Key Features:**
- 🔐 JWT-based authentication with Role-Based Access Control (Employee / Manager / Finance / Admin)
- 🤖 Tesseract OCR receipt scanning with 90%+ data extraction accuracy
- 🚨 AI fraud detection engine with duplicate receipt detection (SHA-256 hashing)
- 📊 Real-time expense tracking with multi-level approval workflow
- 📧 Automated email notifications via Spring Mail + Gmail SMTP
- 📈 Management analytics with department-wise and category-wise reports
- 🏗️ Feature-based MVC architecture with 8 independent modules
